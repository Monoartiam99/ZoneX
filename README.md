# HostedGames UI Demo

This is a small static UI prototype demonstrating the home, tournaments, wallet, leaderboard and admin pages described in your brief. It's intentionally lightweight so you can open it locally and iterate.

How to open
- Open `d:/Programming/HostedGames/website-demo/index.html` in your browser.

What included
- `index.html` — Home page demo
- `tournaments.html`, `tournament.html` — Tournament listings and detail
- `wallet.html` — Wallet and transactions demo
- `leaderboard.html` — Leaderboard demo
- `admin.html` — Admin panel layout demo
- `css/styles.css` — Theme variables and styles
- `assets/` — logo and placeholder images

Color theme
- Primary: purple (#6C4CE0)
- Accent: green (#00C853)
- Accent2: yellow (#FFD600)
